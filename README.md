# 🎬 Movie Explorer (React App)

A React movie explorer web application that fetches and displays movie data using an API.
Built using React, Vite, Axios, and CSS.

## 🚀 Features

* Fetch movies from API
* Display movie cards
* Loading state
* Error handling
* Clean UI using CSS
* Responsive layout

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* CSS
* Axios
* REST API

## 📂 Project Structure

```
src/
 ├── components/
 │     ├── fetch.jsx
 │     ├── fetch.css
 ├── App.jsx
 ├── main.jsx
```

## ▶️ Run Locally

```
npm install
npm run dev
```
